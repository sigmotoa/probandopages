Comandos de inicio

  1. git init -> Inicializa el repositorio de forma local
  2. git status -> Me muestra el estado del repositorio (cosas pendientes o listas)
  3. git add "file/s" -> Adiciona los archivos para que se les haga seguimiento
  4. git commit -m"msg" -> Confirma los cambios adicionados
